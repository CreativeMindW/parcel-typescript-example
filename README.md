# parcel-typescript-example

A minimum TypeScript app with [Parcel Bundler](https://parceljs.org/).

## Usage

### Installation

    yarn install

### Development

Launch server.

    yarn start
    
### Build

Compile TypeScript app and copy index.html to the `dist` folder.

    yarn run build

### Deploy

Deploy `dist` to GitHub Pages.

    yarn run deploy
